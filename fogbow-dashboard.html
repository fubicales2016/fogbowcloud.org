Fogbow Dashboard

The Fogbow Dashboard is an implemetation of a horizon dashboard and provides a web based user interface to the Fogbow Manager. The operations that can be performed via the Fogbow Dashboard are the same provided by the Fogbow CLI.

Installation

git clone https://github.com/fogbow/fogbow-dashboard.git
cd fogbow-dashboard
# Download libraries and run tests
./run_tests.sh

Configure

After the installation, edit the local_settings.py file to point the Fogbow Dashboard to the HTTP endpoint of a Fogbow Manager:

# Fogbow Manager to be used.
FOGBOW_MANAGER_ENDPOINT = "http://localhost:8182"

Run

./run_tests.sh --runserver localhost:9000

To know more about the Horizon project, and how to deploy it within different environments, see the Horizon docs.
